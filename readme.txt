1003 Mortgage Application provides Affordable Mortgages and Award-Winning Client Service.

== Description ==

1003 Mortgage Application is a very easy-to-use wordpress plugin built with the purpose of providing Affordable Mortgages and other Award-Winning Client Service.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. WP installations have the settings available under the "1003 Mortgage Application" menu item.
4. Configure setting and outbound email settings here from the General Settings tab.
5. Next you can configure each of the Application settings individually, save them and make sure to test to ensure everything is coming through as you desire.
6. Once all the settings are configured you will copy the shortcode from the top of each 1003 Mortgage Application settings screen and place it on a page of your choice to show the Mortgage Application Form (ex. [mortgage_application_form])

== Frequently Asked Questions ==

= Is this plugin free forever? =

Yes you will get the Freemium model absolutely free forever and receive our killer support and updates on a regular basis to ensure it stays compatible with the latest version of WP.

= Can I edit the look of the calculator? =

Yes you can. We have provided a setting menu that will change the look/feel of your application simply by choosing the settings.
